Tic Tac Chess

Tic Tac Chess is a two-player strategy game that combines the logic of Tic Tac Toe with the movement mechanics of chess. Each player controls chess pieces, aiming to align their three chosen pieces in a row to win. Unlike traditional chess, no pieces can be captured, so every move requires careful planning and foresight. I developed the game in C# (Windows Forms), handling turn-based logic, valid chess movement rules, and win detection. Later, I connected it to an Arduino-powered robot arm that could physically move the pieces on a real board. This project deepened my skills in object-oriented programming, game logic design, and integrating hardware with software.

quizmester

Quizmester is a C# quiz application that allows users to register, log in, and play timed quizzes from a database. Players can choose categories, answer randomized multiple-choice questions, and compete for a spot on the top 10 leaderboard. The app includes a full CRUD system for an administrator to manage users and questions. I built this project using C# and MySQL. It strengthened my understanding of database integration, user authentication, and object-oriented programming in C#.
